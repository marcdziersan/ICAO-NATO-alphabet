# About this project

This app was created out of a personal interest in aviation, maritime communication and clear radio language. The ICAO/NATO/IMO alphabet is one of the most practical examples of how standardized wording can improve safety and understanding.

## Why this project exists

- make the spelling alphabet easier to learn interactively
- combine visual learning with speech output
- keep the app lightweight and browser-based
- provide a simple, maintainable example of a small PWA-style project

## Focus areas

- accessibility and clarity
- modern browser support
- low-maintenance, dependency-free code
- room for future extensions such as quizzes or saved progress
